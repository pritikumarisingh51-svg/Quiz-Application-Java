# Quiz Application (Java)

## 🚀 Features
- Multiple-choice questions
- Interactive GUI using Java Swing
- Score calculation
- User-friendly interface

## 🛠️ Technologies Used
- Java
- Swing (GUI)
- ArrayList (Collections)
- OOP Concepts

## ▶️ How to Run
1. Compile: javac *.java
2. Run: java QuizGUI

## 📌 Description
This project is a simple quiz application built using Java. It displays multiple-choice questions, takes user input, and calculates the final score.
